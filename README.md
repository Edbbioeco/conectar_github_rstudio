# Conectando o RStudio ao GitHub


> Guia prático para usuários de R que não são da área de tecnologia

# Pacotes necessários

> [!IMPORTANT]
>
> Antes de carregar os pacotes, certifique os mesmos estão previamente
> instalados. Caso não, instale-os usando a função `install.packages()`.

``` r
library(usethis)

library(gitcreds)

library(gert)
```
